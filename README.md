# double-blind-review

Anonymized peer review platform with structured rubrics, reviewer calibration, and inter-rater reliability analysis (Krippendorff's alpha).

## Quickstart

```bash
git clone https://github.com/abhi-wadhwa/double-blind-review.git
cd double-blind-review
make install
make run
make test
```

See `docs/` for the math.
